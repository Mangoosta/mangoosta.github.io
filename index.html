La empresa

Mangoosta es una empresa "orientada a datos". Somos personas con experiencia en manejo, procesamiento, análisis, desarrollo y despliegue de aplicaciones orientadas a datos, y este es nuestro elemento diferenciador.

En Mangoosta, ¡el rey es el dato!

Nuestro Propósito

Las personas son entidades andantes de información. Queremos interconectar esas personas y la información que generan, haciéndola significativa y útil para propender a una nueva sociedad global del conocimiento.

 
accessibility

Nuestra Visión

Crear una nueva sociedad global del conocimiento, posible, no utópica, basada en información práctica, útil, precisa, redárquica, justa y adecuadamente distribuida, que permita empoderar tanto a personas como a empresas en general, siempre pensando en el aporte social global.

 
nature_people

Nuestra Naturaleza

Así como la mangosta (Herpestidae) puede enfrentarse a otros animales del doble de su tamaño, es totalmente gregaria, se comunica con otros mediante la simplificacíón de redes complejas de comunicación y posee una agilidad sorprendente. Así somos nosotros.


Nos apasiona la tecnología...

Back-end

Utilizamos C++ 11 con las librerías más actualizadas para IoT, sistemas en tiempo real y transpilación

Trabajamos con el paradigma NoSQL, usando para ello las mejores herramientas, como CouchDB y/o MongoDB

Usamos Java para desarrollo de aplicaciones nativas que requieran altos volúmenes de datos

PHP es un lenguaje de servidor ampliamente utilizado en aplicaciones web

Python es un lenguaje muy útil en el procesamiento de data y el análisis de datos

En aquellos casos en los que un modelo relacional es requerido, confiamos en herramientas como PostgreSQL y/o MySQL

Dev-ops

Docker es muy confiable en aquellos entornos donde se requiera un despliegue rápido de aplicaciones

Nuestra herramienta de control de versiones favorita. Versátil para gestión de proyectos y entrega contínua

Jenkins es ampliamente utilizado en integración continua por su capacidad de interoperación con otras herramientas como Gitlab

Sonarqube es una excelente herramienta para control de calidad de código fuente, entre otras alternativas

Mangoosta es una empresa orientada a los datos. Nada mejor para describirlos y clasificarlos que Swagger.io

Apache Spark es actualmente la plataforma de Big Data más utilizada en el mundo. ¡Y también nosotros!

Front-end

Angular.js se ha convertido en el Framework de referencia para desarrollo de aplicaciones orientadas a datos

Podemos desarrollar aplicaciones con un nivel de diferenciación importante usando HTML5 y CSS3. ¡Como está página web!

Ionic Framework es ampliamente usado en la industria de desarrollo de software para aplicaciones híbridas

Node.js es nuestra base siempre confiable sobre la cual desarrollamos nuestras interfaces de aplicaciones

¿Requiere de un nivel de personalización muy específico para su aplicación? Sass nos ayuda mucho con eso

Socket.io es una librería muy confiable y ampliamente utilizada cuando se trata de desarrollar aplicaciones y sistemas en tiempo real
